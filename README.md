# AI-Based-Health-Advisory-System-using-ML-DL-for-CKD-prediction-and-healthcare-recommendations.
The **AI-Based Health Advisory System Using Machine Learning and Deep Learning** is an intelligent healthcare monitoring and prediction solution designed to support healthcare decision-making by analyzing patient medical data and generating predictive insights. Healthcare systems often face challenges in early disease detection and personalized health recommendations, which can lead to delayed treatment and increased health risks. This project aims to address these challenges by using Artificial Intelligence technologies to assist in disease prediction and health advisory support.

The system is developed using **Machine Learning (ML) and Deep Learning (DL)** techniques to analyze patient medical information and predict potential health conditions, particularly **Chronic Kidney Disease (CKD)**. By collecting and processing patient health parameters in real time, the system helps identify possible health risks at an early stage and supports effective healthcare decision-making.

To achieve this, the project integrates **data preprocessing, feature analysis, predictive modeling, and intelligent healthcare analytics** using ML and DL algorithms. A **web-based application** was developed to allow users to enter patient medical information, monitor prediction results, and receive healthcare recommendations based on predictive analysis. The system is designed to improve accuracy in health prediction and provide a user-friendly experience for healthcare monitoring.

The project also includes **health data analysis and intelligent prediction mechanisms** that assist users and healthcare professionals in identifying medical conditions based on historical and current patient information. By leveraging AI technologies, the system supports faster analysis, better healthcare insights, and improved patient monitoring.

This solution focuses on creating a **smart, reliable, scalable, and data-driven healthcare advisory system** that can support disease prediction, improve preventive healthcare, and enhance medical decision-making. By combining **Artificial Intelligence, Machine Learning, Deep Learning, predictive analytics, and web technologies**, the project demonstrates how technology can be used to solve real-world healthcare challenges effectively.

This research work was successfully presented at the **IEEE International Conference on Computing, Communication, Control and Cyber-Physical Systems (I5CPS-2026)**, strengthening practical knowledge in AI-driven healthcare systems and research-based problem solving.
